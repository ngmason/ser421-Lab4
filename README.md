# Vue Jeopardy Game

A multiplayer Jeopardy-style trivia game built with Vue 3. This project demonstrates dynamic UI rendering, API-driven content, player score tracking, and interactive game logic.

## Features

- Supports 2–6 players
- Supports 2–10 trivia categories
- Dynamically retrieves trivia questions from an external trivia API
- Displays a Jeopardy-style game board with category and dollar-value selections
- Assigns question difficulty based on dollar value
- Tracks player turns and scores
- Provides visual feedback for completed questions
- Determines a winner or tie when the game board is completed
- Includes Double Jeopardy wagering mechanics
- Validates wager amounts based on player balance
- Uses a configuration file for category exceptions and game constants

## Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open the local development URL displayed in the terminal.
